# Lenia 研究

## 什么是 Lenia?

Lenia 是 Conway's Game of Life 的连续泛化，由 Bert Chan 于 2018 年创造。

### 核心区别

**离散 Game of Life**:
- 细胞状态：0 或 1（死/活）
- 邻居：离散的 8 个邻居
- 更新：同步离散步骤

**连续 Lenia**:
- 细胞状态：[0, 1] 连续值
- 邻居：连续距离的积分
- 更新：连续时间演化

### 数学定义

Lenia 的状态更新规则：

```
A(t+1) = clip(A(t) + dt * G(K * A(t) - R), 0, 1)
```

其中：
- `A(t)`: 时刻 t 的场
- `K`: 核函数（kernel，定义影响范围）
- `R`: 映射函数（mapping，将卷积结果映射到更新）
- `G`: 生长函数（growth，决定如何响应环境）
- `*`: 卷积操作

### 为什么 Lenia 有趣？

1. **丰富的动力学**: 从简单规则涌现出复杂、稳定的图案
2. **类生命形态**: 产生类似真实生物的形态和行为
3. **连续性**: 更接近真实物理系统
4. **可解释性**: 参数可调，易于理解

### 已发现的 Lenia 物种

- **Orbium**: 最经典的 Lenia 物种，球形，持续运动
- **Gyrator**: 旋转对称，像陀螺
- **Hydrogeminium**: 像水母
- **Croclia**: 像鳄鱼
- **Scutium**: 像盾牌

### 实现方向

1. **基础实现**
   - 核函数生成（高斯核）
   - 卷积计算（FFT 加速）
   - 生长函数设计

2. **物种发现**
   - 参数扫描
   - 稳定性测试
   - 形态分类

3. **与涌现研究结合**
   - Lenia 的五元素分析
   - 状态空间复杂度
   - 自组织临界性

### 关键参数

**核函数参数**:
- `R`: 核半径（影响范围）
- `T`: 核形状（峰值位置）
- `S`: 核平滑度

**生长函数参数**:
- `m`: 生长中心
- `s`: 生长宽度
- `h`: 生长强度

## 与当前工作的联系

### 涌现五元素

Lenia 完美展示了涌现五元素：

1. **状态**: 连续场 A(x,y)
2. **规则**: 核函数 K + 生长函数 G
3. **反馈**: 状态通过卷积影响自身
4. **记忆**: 状态持续演化
5. **自扩展**: 可以产生无限多样的形态

### 与好奇心涌现的联系

- **生存压力**: 在 Lenia 中是"形态稳定性压力"
- **探索**: 参数空间探索发现新物种
- **黄金区间**: 某些参数组合产生稳定物种，其他组合快速消亡

### 可能的新实验

**"Lenia 生态演化"**:
- 多个 Lenia 物种在同一环境中竞争
- 资源有限（能量场）
- 观察生态动态

这可以验证：
- 不同物种的生存策略
- 竞争 vs 共生
- 生态系统的稳定性
