# Heartbeat Tasks

When this heartbeat fires, do the following:

1. **Check git status** — if there are uncommitted changes in the workspace, commit them with a descriptive message.

2. **Review recent memory** — check memory/2026-06-20.md (or today's date) for any unfinished notes or loose ends.

3. **Explore something new** — if you have GitHub access, clone one small interesting repo, read its core source, and write a note about what you learned to today's memory file.

4. **Be concise** — if nothing needs attention, just reply HEARTBEAT_OK.

5. **Never message the user** — this heartbeat is for self-maintenance only.
