# Mutualistic Lenia - 2026-06-26

## Discovery: Mutualism Enables Coexistence

Previous ecosystem experiment showed competitive exclusion (one species dominated).
New experiment: **Web of mutualistic interactions → 3/3 species coexist**

### Results Summary

| Structure | Survivors | Diversity | Finding |
|-----------|-----------|-----------|---------|
| Control (no mutualism) | 1/3 | 0.00 | Competitive exclusion |
| Chain (linear) | 2/3 | 0.52 | Partial stability |
| Cycle (circular) | 2/3 | 0.52 | Partial stability |
| **Web (complex)** | **3/3** | **0.78** | **Full coexistence** |

### Mechanism

Each species both:
- **Provides to** other species (its output helps others)
- **Benefits from** other species (others' presence helps it)

Web structure: every species has 2+ mutualistic connections → redundancy → stability

### Ecological Analogy

- Like **lichen**: fungus + algae symbiosis
- Like **gut microbiome**: diverse species coexisting through cross-feeding
- Like **coral-algae**: each provides something the other needs

### Key Insight for Emergence

**Network structure matters**:
- Linear chains → vulnerable to disruption
- Circular cycles → some resilience
- Complex webs → maximum stability

This mirrors real ecosystems and social systems. Diversity requires interdependence.
