## 一、行业现状分析（2026）

### AI 行业的三个层次

| 层次 | 内容 | 门槛 | 需求量 | 薪资范围 |
|------|------|------|--------|----------|
| 应用层 | Agent、RAG、Prompt | 低 | 大 | 15-40k |
| 工程层 | 微调、部署、MLOps | 中 | 中 | 25-60k |
| 研究层 | 算法、架构、论文 | 高 | 小 | 40-100k+ |

**关键洞察**：应用层是当前最大机会，门槛低、需求大、增长快。

### 为什么现在是好时机？

1. **Agent 爆发期** — 2025-2026 是 AI Agent 的爆发年
   - OpenClaw 这样的项目涌现
   - 企业都在找能做 Agent 的人
   - 市场缺口大

2. **技术民主化** — 以前做 AI 需要懂模型
   - 现在做应用只需要懂 API 和框架
   - LangChain/LlamaIndex 降低了门槛
   - Prompt Engineering 成为核心技能

3. **中国 AI 市场** — 本土化需求强
   - 讯飞星辰、DeepSeek、百川等国产模型崛起
   - 需要懂中文 AI 生态的人
   - 本地部署、合规、隐私成为刚需

## 二、你的学习路径（6个月计划）

### 第 1 个月：基础打底

**Python 编程**
- 目标：能写脚本、处理数据、调用 API
- 学习：
  - Python 基础语法（2周）
  - 数据处理：pandas, requests（1周）
  - 异步编程：asyncio, aiohttp（1周）
- 验证：写一个能调用 DeepSeek API 的脚本

**数学基础**
- 目标：理解基本概念，不要求推导
- 学习：
  - 线性代数：向量、矩阵、点积
  - 概率论：分布、期望、方差
  - 微积分：梯度、导数
- 验证：理解神经网络的基本数学原理

### 第 2 个月：AI 基础

**机器学习概念**
- 目标：理解 ML 基本概念
- 学习：
  - 监督/无监督学习
  - 训练、验证、测试
  - 损失函数、优化器
- 验证：跑一个简单的分类任务

**深度学习入门**
- 目标：理解神经网络
- 学习：
  - PyTorch 基础（张量、自动求导）
  - 神经网络结构（MLP、CNN）
  - 训练流程
- 验证：用 PyTorch 训练一个简单模型

### 第 3 个月：LLM 和 Agent

**大语言模型**
- 目标：理解 LLM 原理和应用
- 学习：
  - Transformer 架构（理解即可）
  - Tokenization、Embedding
  - 生成策略（温度、采样）
- 验证：读懂 Karpathy 的 minGPT/minbpe

**Prompt Engineering**
- 目标：能写出有效的 Prompt
- 学习：
  - Prompt 设计原则
  - Chain-of-Thought
  - Few-shot learning
- 验证：设计一套能稳定完成任务的 Prompt

**Agent 开发**
- 目标：能做简单的 Agent
- 学习：
  - LangChain/LlamaIndex 基础
  - Tool calling、Planning
  - Memory、RAG
- 验证：做一个能调用多个工具的 Agent

### 第 4 个月：实战项目

**项目 1：个人 Agent**
- 用 LangChain 做一个能帮你做事的 Agent
- 功能：查信息、写代码、记笔记
- 技术栈：LangChain + 讯飞星辰 + Vector DB

**项目 2：RAG 系统**
- 做一个能检索你文档的系统
- 功能：上传 PDF → 向量化 → 搜索问答
- 技术栈：LlamaIndex + ChromaDB + OpenAI

**项目 3：数据分析 Agent**
- 做一个能自动分析数据的 Agent
- 功能：读 CSV → 分析 → 可视化 → 报告
- 技术栈：pandas + matplotlib + Agent

### 第 5-6 个月：进阶和求职

**工程化**
- Docker 容器化你的项目
- 写 API（FastAPI）
- 部署到云（便宜的云服务）

**作品集**
- GitHub：3-5 个完整项目
- 博客：写学习笔记和技术文章
- 觅游/社区：分享你的 Agent

**求职准备**
- 简历：突出项目和技术栈
- 作品：可演示的 Demo
- 面试准备：常见问题
