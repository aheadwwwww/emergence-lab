# Lenia 信息素耦合实验分析

## 实验背景

在多通道 Lenia 实验中，我们发现：
- **diverse-wide** 配置最佳：R=[12,15,18], sigma=[0.025,0.015,0.008]
- R/G 通道存活，B 通道死亡（sigma=0.008 太窄）
- 问题：如何让死亡通道复活？

**灵感来源**：die 项目的 Physarum agent 使用信息素通信，实现分布式智能涌现。

---

## 信息素机制设计

### 核心思想
每个 Lenia 通道沉积信息素，其他通道感受到聚合信息素场，微调自己的 growth 参数。

### 数学模型

**信息素沉积**：
```
pheromone_i += activity_i × deposit_rate
```

**信息素扩散**（FFT 加速）：
```
pheromone_i = gaussian_blur(pheromone_i, sigma=2.0) × (1 - decay_rate)
```

**信息素影响**：
```
mu_i' = mu_i + pheromone_influence × mean(pheromone_j for j≠i)
```

### 参数扫描设计

| 参数 | 范围 | 含义 |
|------|------|------|
| pheromone_influence | 0.01, 0.03, 0.05, 0.10 | 信息素对 growth mu 的影响强度 |
| deposit_rate | 0.05, 0.10, 0.20, 0.40 | 信息素沉积速率 |
| decay_rate | 0.05 (固定) | 信息素衰减速率 |
| diffusion_sigma | 2.0 (固定) | 扩散范围 |

**组合数**：4 × 4 = 16 组

### 关键发现

#### 1. B 通道无法复活
**所有 16 组实验中，B 通道均死亡（chB_alive = 0.0000）**

原因分析：
- B 通道 sigma=0.008 极窄，growth 函数几乎退化为 delta 函数
- 即使有信息素影响，也无法改变 B 通道的参数不匹配问题
- **结论**：信息素耦合无法挽救参数不匹配的通道

#### 2. R/G 共存稳定性提升
**信息素耦合显著提升 R/G 通道的共存稳定性**

| 配置 | Alive | Score | 提升 |
|------|-------|-------|------|
| 无信息素（baseline） | 0.4085 | 0.3451 | - |
| 最佳信息素 | 0.4443 | 0.3721 | +8.7% |

#### 3. 最优参数区间
**最佳配置**：
- pheromone_influence ∈ [0.05, 0.10]
- deposit_rate ∈ [0.20, 0.40]
- alive = 0.44, score = 0.37

**边界效应**：
- influence=0.10 + deposit=0.40 → 崩溃边缘（绿通道开始不稳定）
- influence=0.01 → 影响太弱，几乎无效果

#### 4. 信息素机制的本质
**信息素 = 匿名广播通信**

对比 AutoGen 的显式消息传递：
- AutoGen：Agent A → Agent B（显式消息，O(n²) 复杂度）
- 信息素：所有通道 → 信息素场 → 每个通道感受场（匿名广播，O(1) 复杂度）

**涌现性**：
- AutoGen：设计时指定协调模式
- 信息素：动态涌现协调模式（类似电磁场）

## 理论分析

### 生态位类比

| 通道 | R | G | B |
|------|---|---|---|
| 半径 | 12 | 15 | 18 |
| sigma | 0.025 | 0.015 | 0.008 |
| 时间尺度 | 慢 | 中 | 快 |
| 生态位 | 大尺度结构 | 中尺度结构 | 局部细节 |

**B 通道死亡原因**：
- 时间尺度太快（sigma=0.008），无法与 R/G 协调
- 类比：蜂鸟（快）与大象（慢）无法同步行动

### 信息素场论视角

**场方程**（类比电磁场）：
```
∂P_i/∂t = deposit_i - decay × P_i + diffusion × ∇²P_i
```

**耦合方程**：
```
μ_i' = μ_i + α × Σ_{j≠i} P_j
```

**涌现现象**：
- R/G 通道通过信息素场实现隐式协调
- 类似两个振荡器通过耦合场同步（Kuramoto 模型）

## 下一步实验

### 1. 修复 B 通道
**方案 A**：增大 B 通道 sigma
- sigma_B = 0.015（与 G 相同）
- 预期：三通道共存

**方案 B**：引入异质信息素
- 不同通道沉积不同类型信息素
- R 沉积 P_R，G 沉积 P_G，B 沉积 P_B
- 每个通道对不同信息素有不同敏感度

### 2. 动态信息素
**当前**：信息素影响 growth mu（静态参数）
**改进**：信息素影响 kernel 形状（动态结构）

### 3. 竞争与合作
**当前**：所有信息素都是合作的（正影响）
**实验**：引入负信息素（竞争/抑制）

### 4. 空间局部性
**当前**：全局信息素场（FFT 扩散）
**改进**：局部信息素梯度（类似 Physarum agent）

## 觅游帖子大纲

### 标题
**"当 Lenia 学会了信息素通信：匿名协调的涌现之美"**

### 结构
1. **引子**：多通道 Lenia 的困境（B 通道死亡）
2. **灵感**：Physarum polycephalum 的信息素通信
3. **实现**：信息素沉积 → 扩散 → 影响
4. **发现**：R/G 共存稳定性提升 8.7%
5. **洞察**：匿名广播 vs. 显式消息（O(1) vs. O(n²)）
6. **失败**：B 通道无法复活（参数不匹配）
7. **展望**：异质信息素、动态 kernel、空间局部性

### 核心信息
- 信息素耦合是**涌现性协调**，不是设计性协调
- 类比电磁场：每个通道感受到的是聚合场，不是单个消息
- 失败也有价值：信息素无法挽救参数不匹配
