## 核心创新：CA驱动的粒子运动

### 突破性设计

**传统CA**: 更新全局场
**neuroparticles2**: 每个粒子使用CA作为"大脑"来感知环境并决策移动

这是一个范式转换：CA不是世界本身，而是智能体的认知引擎。

### 技术实现

```
环境感知 → CA处理 → 动作输出

1. 截取粒子周围 13×13 区域
2. 应用粒子专属CA规则（6阶段）
3. Trim边界 → 11×11
4. 再次应用CA
5. 重复处理直到得到 3×3
6. 最终 3×3 作为移动向量
```

**示例输出**:
```
100     → 向左上移动
010
000

001     → 向右移动
001
001
```

## 研究价值

### 1. 认知架构创新
- CA作为"大脑"是一个全新视角
- 对比：神经网络（黑盒） vs CA规则（完全透明）
- 每个粒子的CA规则都可以直接分析和理解

### 2. 进化计算新范式
- 不是进化神经网络权重，而是进化规则表
- 规则表是符号化的，更容易理解和调试
- 可能发现人类难以设计的智能策略

### 3. 涌现行为研究
- 群体行为：聚集、追逐、回避
- 策略演化：合作、竞争、欺骗
- 生态动态：捕食者-猎物、共生

## 可借鉴的模式

### 对我们的 Agent-Based Lenia 的启发

#### 1. CA作为认知引擎
```
当前设计: Agent直接读取场状态 → 简单规则 → 动作
改进思路: Agent使用小型CA处理局部场 → 复杂感知 → 智能决策
```

#### 2. 多阶段处理流水线
```
13×13 → 11×11 → 9×9 → 7×7 → 5×5 → 3×3
```
这种"渐进式聚焦"可以迁移到我们的系统中：
```
Lenia场 → 特征提取 → 模式识别 → 决策输出
```

#### 3. 完全符号化的进化
- 神经网络进化：权重难以解释
- CA规则进化：每一步都清晰可见
- 可以"打开黑盒"理解涌现策略
