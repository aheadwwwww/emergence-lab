# Die 信息素机制 → Lenia 耦合

## 从 Die 学到的核心设计

### 信息素场 (Stigmergy)
Die 的核心创新：agent 释放信息素标记食物位置，其他 agent 趋向信息素浓度高的区域。

关键实现细节（从 `core/env.py` 和 `core/agent/gradient.py`）：
- **沉积**：agent 在当前位置沉积化学物质 (`deposit1` channel)
- **扩散+衰减**：`filters.gaussian(sigma=0.5)` 扩散 + `rate_decay_chem=0.1` 衰减
- **梯度感知**：agent 计算化学场的梯度来决定移动方向
- **惯性**：`inertia=0.9` 让 agent 保持之前的方向，避免过度反应

### 与 Lenia 的类比

| Die 概念 | Lenia 类比 |
|----------|-----------|
| 化学场 (chem1) | 通道间耦合场 |
| 扩散+衰减 | kernel 卷积 + growth 衰减 |
| 梯度感知 | 通道间通过耦合矩阵感知 |
| 惯性 | 时间步 dt 的平滑效果 |

## 实现思路：信息素耦合 Lenia

在已有的多通道 Lenia 基础上加入：
1. **信息素通道**：额外的"通信场"，每个通道可以写入和读取
2. **信息素扩散**：独立的高斯扩散（不同于 kernel 卷积）
3. **信息素衰减**：每步衰减一定比例
4. **通道间信息素梯度**：通道根据信息素梯度调整自己的 growth 参数

## 实验设计

1. 3 个 Lenia 通道 + 3 个信息素通道（每个 Lenia 通道对应一个信息素通道）
2. 信息素沉积 = 通道活性 × deposit_rate
3. 信息素扩散：sigma=2.0, decay=0.05
4. 通道根据信息素梯度微调 mu（growth center）

预期：信息素耦合可能产生更稳定的多物种共存，类似生态位分化。
