# #024 Predictive Processing — 感知即受控幻觉

## 核心概念

**Predictive Processing**（预测加工/预测编码）认为大脑不是被动接收感官信息，而是一个**主动预测引擎**：

1. **自上而下的预测**：大脑持续生成对感官输入的期望
2. **自下而上的误差**：当预测与实际输入不匹配时，产生预测误差
3. **误差最小化**：大脑更新内部模型以最小化预测误差
4. **精确性加权**：对"可信"的信号给予更高权重

> "Perception is controlled hallucination" — Richard Gregory

## 关键关联

- **#001 Emergence**：感知是预测的涌现结果，不是输入的简单映射
- **#015 Cognitive Gap**：认知鸿沟部分源于预测模型的先验偏差
- **#016 Self-Reference**：自我建模可以看作是最高层的预测编码
- **#018 Hallucination**：当先验完全压制感官输入时 = 幻觉
- **#019 Strange Loops**：预测→感知→更新预测→... 感知本身是一个自指循环
- **#012 Scaling Laws**：更大的模型容量 = 更好的预测 = 更丰富的感知
- **#017 Creativity**：创造性 = 故意偏离预测，探索"错误"的可能性

## 实验

### 1. 层级预测编码 (`predictive_encoding.png`)
- 2层系统：高层学习正弦波频率，低层生成具体预测
- 频率估计从初始猜测逐步收敛到真实值
- 预测误差随时间下降（但噪声导致持续波动）

### 2. 模糊刺激 (`ambiguous_perception.png`)
- 两个模式模糊混合的相同输入
- 不同先验偏差→完全不同的"感知"
- 先验偏差60%可让感知完全偏向一方
- 先验-感知映射是S形曲线（非线性）

### 3. 精确性加权 (`precision_weighting.png`)
- 4段信号：σ=[0.1, 0.8, 0.2, 1.2]
- 精确性高的段（σ=0.1）后验不确定性迅速下降
- 噪声段（σ=1.2）大脑保持"不确定"——不盲目相信模糊输入
- Kalman Gain = 学习率，随精确性自适应调整

### 4. 层级架构 (`predictive_hierarchy.png`)
- 高层→低层：预测（向下）
- 低层→高层：预测误差（向上）
- 公式：Perception = argmin(Prediction Error)

## 关键洞察

1. **感知不是输入的处理，而是预测的检验**。大脑看到的是自己"认为"在那里的事物，用感官数据来微调。

2. **模糊性是常态，不是例外**。大脑总是在用先验补完不完整的感官输入。

3. **"受控幻觉"的"控制"来自精确性**——当输入可靠时，先验让步；当输入模糊时，先验主导。

4. **创造力与幻觉在同一光谱上**：两者都是先验主导感知，区别在于"控制"的程度。

5. **预测编码是涌现的计算框架**：局部误差信号→全局模型更新→涌现出感知能力。
