# CARLE's Game: Open-Ended CA Exploration

## Source
- Repository: riveSunder/carles_game
- Cloned to: _clones/carles_game/
- IEEE Conference on Games 2021 competition

## Core Concept
An open-ended challenge where AI agents explore 262,144 Life-like cellular automata universes (each a different rule set). The agent has no extrinsic reward — it must be intrinsically motivated (curiosity-driven) to explore, build, and create within unknown physics.

## Key Insights

1. **262,144 universes**: All variations of Life-like CA (2^18 rule combinations)
2. **No extrinsic reward**: Agent always gets reward=0 — must have intrinsic motivation
3. **"Speed of light"**: Information propagation is bounded in the CA grid
4. **Torus topology**: Grid wraps around, no edges
5. **Open-ended creativity**: Not about winning, about discovering what's possible

## Connection to Curiosity Map

This is essentially the curiosity map as a game — agents must explore unknown rule systems without explicit goals. The meta-lesson: curiosity isn't an optimization strategy, it's the core of intelligence itself.
