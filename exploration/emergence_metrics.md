# Emergence Metrics & Complexity Theory

## The Challenge: Quantifying Emergence

How do we measure when something "emerges"? When does a system become more than the sum of its parts?

## Key Metrics

### 1. Integrated Information (Φ) - IIT
**Theory**: Tononi's Integrated Information Theory
- Measures how much information is generated by a system as a whole, beyond its parts
- High Φ = system is irreducible, has "consciousness-like" unity
- **Formula**: Φ = minimum information partition (MIP) complexity

**Application to Experiments**:
- Boids flock: High Φ when coherent, low when scattered
- Game of Life glider gun: High Φ (sustained structure)
- Sandpile: Moderate Φ during avalanche, low at rest

**Tools**: 
- pyphi library (Python)
- Compute Φ for small systems (8-16 nodes max, computationally expensive)

### 2. Statistical Complexity (Crutchfield)
**Theory**: Computational Mechanics / Epsilon-Machine
- Minimum amount of information needed to predict future states
- Captures "hidden structure" in dynamics
- **C_r = H[States of Causal States]**

**Application**:
- Wolfram CA: Class 4 has highest statistical complexity
- Langton's Ant: Cr high when building highway, low in chaos

### 3. Multi-Scale Entropy
**Theory**: Entropy at different spatial/temporal scales
- Sample Entropy, Approximate Entropy, Multiscale Entropy
- Measures complexity across scales

**Application**:
- Turing Patterns: High entropy at small scale, low at large (pattern!)
- Boids: Scale-dependent entropy reveals flock vs individual

### 4. Causal Emergence (Hoel)
**Theory**: Higher scales can have MORE causal power than lower scales
- **Effective Information (EI)** at each scale
- Causal emergence: EI(higher) > EI(lower)
- Quantifies when macroscale is MORE deterministic than microscale

**Application**:
- Ising Model: At critical T, magnetic domains have more causal power than spins
- Game of Life: Glider has more causal power than cells

### 5. Synergistic Information
**Theory**: Information that exists ONLY in the whole, not in parts
- Partial Information Decomposition (PID)
- **Synergy > 0** indicates emergence

**Formula**: I(X:Y) = Unique(X) + Unique(Y) + Redundant(X,Y) + Synergistic(X,Y)

### 6. Transfer Entropy
**Theory**: Directed information flow
- Measures how much past of X predicts future of Y beyond Y's own past
- **T_{X→Y}** = information transfer from X to Y

**Application**:
- Boids: T_{leader→follower} reveals leadership emergence
- Neural-like CAs: Information cascades

## Practical Implementation for Orchestrator

### Tier 1: Fast Metrics (All Experiments)
1. **Shannon Entropy**: Fast, baseline complexity
2. **Spatial Correlation Length**: Phase transition indicator
3. **Cluster Size Distribution**: Power law = criticality

### Tier 2: Medium Metrics (Select Experiments)
1. **Statistical Complexity**: Via CSSR algorithm
2. **Transfer Entropy**: For Boids, Lenia
3. **Multi-Scale Entropy**: For spatial patterns

### Tier 3: Deep Metrics (Small Systems Only)
1. **Integrated Information (Φ)**: For small networks, Lenia patterns
2. **Causal Emergence**: Via effective information computation

## Code Example: Computing Emergence Metrics

```python
import numpy as np
from scipy.stats import entropy
from scipy.ndimage import label

def compute_emergence_metrics(grid):
    """Compute multiple emergence metrics for a 2D grid"""
    
    # 1. Shannon Entropy
    hist, _ = np.histogram(grid.flatten(), bins=20, density=True)
    H = entropy(hist + 1e-10)
    
    # 2. Spatial Correlation
    from scipy.signal import correlate2d
    corr = correlate2d(grid, grid, mode='same')
    corr_len = np.std(corr)  # Proxy for correlation length
    
    # 3. Cluster Size Distribution
    labeled, num = label(grid > np.mean(grid))
    sizes = [np.sum(labeled == i) for i in range(1, num+1)]
    if len(sizes) > 1:
        sizes = np.array(sizes)
        # Check for power law
        from scipy.stats import linregress
        log_sizes = np.log(np.sort(sizes)[::-1])
        log_ranks = np.log(np.arange(1, len(sizes)+1))
        slope, _, r_value, _, _ = linregress(log_ranks, log_sizes)
        power_law_r2 = r_value ** 2
    else:
        power_law_r2 = 0
    
    return {
        'entropy': H,
        'correlation_length': corr_len,
        'num_clusters': num,
        'power_law_r2': power_law_r2
    }
```

## Theoretical Questions

1. **Is there a universal "emergence threshold"?**
   - Phase transitions have critical points
   - Do all emergent systems cross a critical Φ or Cr?

2. **Scale-dependence**
   - At what scale does emergence appear?
   - Multiscale analysis needed

3. **Dynamics vs Statics**
   - Many metrics are static
   - How to capture dynamic emergence (pattern formation)?

4. **Computational Tractability**
   - Φ is exponentially expensive
   - Need approximations for real systems
