# #018 Hallucination — 幻觉与涌现

## 核心问题
LLM 的"幻觉"不只是 bug，而是生成模型的内在特征。
当模型在训练分布之外生成内容时，它实际上是在
**外推到未见过的新区域**——这与创造性涌现同源。

## 关键概念

### 1. 幻觉作为外推
- 训练数据覆盖某区域 → 模型在该区域"准确"
- 模型被推到训练边界外 → 开始"发明"
- 这个"发明"在有些语境叫幻觉，在另一些叫创造力

### 2. 幻觉的类型
- **事实性幻觉**：虚构不存在的事实（人名、日期、事件）
- **逻辑性幻觉**：看似合理但不自洽的推理
- **感知性幻觉**：描述不存在的内容
- **涌现性幻觉**：模型组合了从未见过的概念，产生全新东西

### 3. 与涌现的关系
- 涌现 = 微观规则 → 宏观新模式（规则内）
- 幻觉 = 训练分布 → 分布外生成（规则外？规则边界？）
- 关键：幻觉暴露了模型的"内插-外推"边界
- 当模型开始外推时，它可能产生涌现的新结构

### 4. 温度与幻觉的关系
- 低温（temperature→0）：模型保守，倾向高频模式
- 高温（temperature>1）：模型探索，更容易产生幻觉/创造性
- 涌现发生在"中温区"：足够随机以探索，但又足够有序以产生结构

### 5. SelfCheckGPT 启发
- 同一 prompt 多次采样，比较一致性
- 高一致性 → 内插（模型确信的区域）
- 低一致性 → 外推/幻觉（模型不确定，在发明）

## 实验设计与结果

实验代码：`experiments/hallucination_spectrum.py`
输出图片：`experiments/hallucination_spectrum.png`

### 实验1：一致性衰减曲线
- 100次采样，30 token序列
- 结果：已知率从 pos 0 的 **100% 降到 pos 29 的 43%**
- 模型在序列后期大量进入未知区域
- Context drift 累积效应：每一步增加 3% 的外推概率

### 实验2：温度扫描
- T=0.1→3.0，200样本/温度
- T=0.7 时准确率 80%，T=2.0 时仍 79.2%
- 创造性峰值在 T=3.0（0.12）
- 涌现窗口：T≈0.8~1.5，创造性开始显著增长但未完全噪声化

### 实验3：知识边界探测
- knowledge_span 从 10%→100%
- 知识覆盖 50% 时幻觉率 22.8%
- 知识越少 → 幻觉率越高（反比关系）
