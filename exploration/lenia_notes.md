# Lenia 学习笔记

## 什么是 Lenia？

Lenia 是康威生命游戏（Conway's Game of Life）的"平滑"连续版本，由 Bert Chan 在 2018 年创建。

## 核心区别

### 康威生命游戏（离散）
- 细胞状态：0 或 1（死/活）
- 邻居：8 个相邻细胞
- 规则：确定性（B3/S23）
- 步骤：离散更新

### Lenia（连续）
- 细胞状态：0 到 1 的连续值
- 邻居：环形核函数（高斯加权）
- 规则：连续增长函数
- 步骤：平滑演化

## 数学原理

### 1. 核函数（Kernel）
```
K(r) = exp(-((d/R - r)²) / σ²)
```
- R: 核半径
- r: 环形核的中心位置（0.35-0.45）
- σ: 控制环的宽度

### 2. 增长函数（Growth Function）
```
G(u) = 2 * exp(-(u - μ)² / (2σ²)) - 1
```
- u: 邻域加权和
- μ: 最佳密度（~0.13）
- σ: 容差范围（~0.015）

### 3. 更新规则
```
A(t+1) = clip(A(t) + dt * G(K * A(t)), 0, 1)
```
- * 表示卷积操作
- dt: 时间步长（通常为 1）

## 涌现行为

Lenia 可以产生丰富的生命形式：
- **Glider**: 类似生命游戏的滑翔机
- **Orbium**: 最著名的"物种"，稳定的移动模式
- **Gyrator**: 旋转结构
- **Hydra**: 多头结构

## 参数空间探索

关键参数：
- **R**: 核半径（10-13）
- **r**: 核中心（0.35-0.45）
- **m (μ)**: 最佳密度（0.12-0.14）
- **s (σ)**: 增长容差（0.012-0.02）

参数空间非常大，只有小区域产生稳定模式。

## 实现要点

### 卷积优化
使用 FFT 可以将卷积从 O(n²) 降到 O(n log n)：
```python
from scipy import fft
kernel_fft = fft.fft2(kernel)
state_fft = fft.fft2(state)
new_state = fft.ifft2(kernel_fft * state_fft).real
```
