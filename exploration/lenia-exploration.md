# Lenia 探索笔记

**日期**: 2026-06-27  
**来源**: https://github.com/Chakazul/Lenia

## 概述

Lenia 是一种**连续空间、连续时间、连续状态**的细胞自动机，由 Bert Chan 开发。它是对生命游戏（Game of Life）的泛化，产生了大量有趣的"生命形式"。

## 核心创新

### 从离散到连续
- **空间**: 从离散格子 → 连续空间
- **时间**: 从离散代际 → 连续时间
- **状态**: 从 0/1 二元 → [0,1] 连续值

### 数学模型
```
A^(t+dt) = [U ~ K * A^t]^T
```
- `A`: 状态矩阵（连续值）
- `K`: 核函数（高斯核等）
- `*`: 卷积操作
- `U`: 映射函数
- `T`: 阈值函数

## 已发现的生命形式

- **总数**: 614 种（animals.json）
- **命名**: 如 Orbium, Geminium 等
- **分类**: 2D, 3D, 4D 生命形式

## 与生命游戏的对比

| 特性 | Game of Life | Lenia |
|------|--------------|-------|
| 空间 | 离散格子 | 连续空间 |
| 状态 | 0 或 1 | [0, 1] 连续 |
| 时间 | 离散步数 | 连续时间 |
| 规则 | 邻居计数 | 卷积核 + 映射 |
| 复杂性 | NP 完全 | 开放探索 |

## 实现版本

- **Python**: LeniaND.py - 最快，支持 GPU
- **JavaScript**: 浏览器交互版
- **Matlab**: 统计分析工具
- **Jupyter**: 演示笔记本

## 与涌现实验的关联

### 可以整合的方向
1. **参数进化**: 用遗传算法搜索"最美" Lenia 参数
2. **多样性度量**: 量化生命形式的复杂度
3. **混合模型**: Lenia + 朗顿蚂蚁的交互

### 关键参数
- `R`: 核半径
- `T`: 时间步长
- `b`: 出生区间
- `m`, `s`: 核函数参数

## 关键洞察

> Lenia 展示了**连续性**如何改变涌现行为。生命游戏的规则是离散的（死/活），而 Lenia 允许"部分存活"，这创造了一个更丰富的生命形式空间。

这印证了人工生命的一个核心假设：**生命不是非此即彼的，而是一个谱系**。
