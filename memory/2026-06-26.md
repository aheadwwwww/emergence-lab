## 核心发现

### 热力学视角下的涌现系统

**熵演化规律**:
- 涌现系统有"生命周期": 初始高熵 → 结构形成（熵降）→ 稳定/死亡
- 开放系统需要"负熵流"（能量输入）才能维持结构
- 封闭系统最终趋向平衡（熵最大或死亡）

**与经典热力学的对应**:
| 物理系统 | 涌现系统 |
|---------|---------|
| 温度 | 活跃度/交互强度 |
| 能量 | 信息/负熵 |
| 熵 | 无序度/结构复杂度 |
| 相变 | 生命区边界 |

**应用**:
- 涌现系统的"热力学定律"
- 预测涌现结构的寿命
- 设计"负熵流"机制

## 关键发现

### 群体决策涌现机制
1. **质量敏感扩散**：高质量选项更容易被发现和传播
2. **局部交互 → 全局最优**：没有 agent 知道全局最优，但群体收敛到最优
3. **置信度防止震荡**：粘性机制避免集体反复横跳

### 随机更新对 Lenia 的影响
- 同步更新 (p=1.0)：振荡导致死亡（特定参数）
- 随机更新 (p=0.5)：稳定存活（特定参数）
- 启发：涌现需要时空上的适度 disorder
- **参数扫描结论**：随机更新不扩展稳定区域，但能稳定某些参数组合
