## 🔬 关键发现

### 能量平衡问题（已解决）

**症状**: Field 崩溃至 0.000，生态系统死亡

**根本原因**: 
- Agents 吸收 > Lenia 再生
- 正反馈循环导致崩溃

**解决方案**:
1. 初始 field 能量提高（0.5-0.8）
2. 吸收上限降低（0.02）
3. 沉积机制（energy > 1.5 时回馈）
4. 平衡 deposit/absorb 比例

**结果**: 系统达到稳态 equilibrium

### 涌现的自调节

```
Agents rich → deposit back → field recovers → more energy available
Agents poor → absorb less → field preserved → sustainable
```

这是 **负反馈回路**，无需编程，从简单规则涌现。
