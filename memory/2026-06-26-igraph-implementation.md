## 三、igraph vs JSON Lines

| 维度 | igraph | JSON Lines |
|------|--------|-----------|
| 存储 | .pkl文件 | .jsonl文件 |
| 查询 | Python API | 文本匹配 |
| 邻居查询 | ✅ node.neighbors() | ❌ 需遍历 |
| 多跳查询 | ✅ get_shortest_paths() | ❌ 需遍历 |
| 性能 | 快 | 慢 |
