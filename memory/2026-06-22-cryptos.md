### 是什么
- Andrej Karpathy 的纯 Python 从零实现比特币

### 关键设计模式

**1. 椭圆曲线点运算重载**
```python
class Point:
    def __add__(self, other):  # 点加法
    def __rmul__(self, k):     # 标量乘法（k * P）
```
使用 secp256k1 曲线：y² = x³ + 7 (mod p)

**2. 密钥派生链**
```
随机 256 位 → 私钥 sk
sk × G → 公钥 pk (椭圆曲线点)
SHA256(pk) → RIPEMD160 → Base58Check → 比特币地址
```

**3. DER 签名编码**
```
0x30 [len] 0x02 [r_len] [r] 0x02 [s_len] [s] [sighash]
```
- 变长整数编码，去除前导零
- 若首字节 ≥ 0x80，需添加 0x00 前缀

**4. Base58Check 编码**
```python
# 地址 = version + hash + checksum
version = b'\x00'  # mainnet
pkb_hash = ripemd160(sha256(pubkey))
checksum = sha256(sha256(version + pkb_hash))[:4]
address = b58encode(version + pkb_hash + checksum)
```

### 学到的技巧

1. **椭圆曲线压缩公钥恢复**
   - 只存储 x 坐标 + y 的奇偶性（前缀 0x02/0x03）
   - 恢复：y² = x³ + 7 (mod p)，取平方根
   - p ≡ 3 (mod 4) 时：y = (y²)^((p+1)/4) mod p

2. **模逆计算**
   ```python
   def inv(n, p):
       return pow(n, p - 2, p)  # 费马小定理
   ```

3. **签名验证数学**
   ```
   给定签名，验证 P = u1*G + u2*pk
   其中 u1 = z*w mod n, u2 = r*w mod n, w = s⁻¹ mod n
   ```

4. **区块难度调整**
   - 每 2016 区块调整一次（约 2 周）
   - 新 target = 旧 target × (实际时间 / 目标时间)
   - Satoshi 有个 off-by-one bug：2015 vs 2016 区块
