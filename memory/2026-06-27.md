### 2. Agent-Based Lenia 原型

创建了 **混合系统原型**（experiments/agent_based_lenia.py）：

**设计思路**：
- 每个 Agent 有 Lenia 参数（R, mu, sigma）作为"基因组"
- 并行操作：Agent 对场的贡献（Lenia 更新）

- 进化：参数突变 + 自然选择

**测试结果**：
- 成功运行 50 步

- Agent 存活、繁殖、死亡机制正常

## 心得

Biomaker CA 的并行/互斥操作分离是一个优雅的设计模式：
- 并行操作无冲突，可以高效执行
- 互斥操作需要仲裁，但保证了竞争的真实性

这个模式可以迁移到很多多智能体系统设计中。

**类比芒格思维模型**：这就像"分工"——明确的角色分离减少了混乱，提高了系统效率。
