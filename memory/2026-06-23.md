### 二、WANN 研究（Weight Agnostic Neural Networks）

**来源**：Google Brain Tokyo Workshop

**核心思想**：
- 神经网络可以完全不依赖权重值来解决问题
- 通过网络拓扑结构（连接方式）来编码智能
- 使用 NEAT（NeuroEvolution of Augmenting Topologies）进化拓扑

**关键机制**：
1. **拓扑进化**：添加节点、添加连接、拓扑变异
2. **权重固定**：所有连接使用相同权重值
3. **多目标优化**：同时优化性能和最小化连接数
4. **物种形成**：保护创新，避免过早收敛

**与涌现研究的关联**：

| 涌现元素 | WANN 对应 |
|---------|----------|
| 状态 | 网络节点 |
| 规则 | 网络连接 |
| 反馈 | 进化压力（适应度） |
| 记忆 | 拓扑结构保留 |
| 自扩展 | 拓扑增长（添加节点/连接） |
| 好奇心驱动 | 物种形成保护多样性 |

**洞察**：WANN 证明了"结构即智能"，这与涌现理论高度一致——简单规则组合出复杂行为，不需要预设权重。
