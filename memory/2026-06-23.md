## 探索：Gensim 主题建模库

### 项目简介
- **Gensim** 是一个 Python 库，专注于**主题建模**、**文档索引**和**相似度检索**
- 目标用户：NLP 和 IR（信息检索）社区
- 当前版本：4.4.0（稳定维护模式，不再接受新功能）

### 核心特性
1. **内存无关性** - 可处理超过 RAM 的大型语料库（流式、out-of-core）
2. **直观接口**
   - 流式 API：轻松接入自定义语料
   - 转换 API：轻松扩展向量空间算法
3. **高效多核实现**
   - LSA/LSI/SVD（潜在语义分析）
   - LDA（潜在狄利克雷分配）
   - RP（随机投影）
   - HDP（层次狄利克雷过程）
   - word2vec 深度学习
4. **分布式计算** - 支持集群运行 LSA 和 LDA

### 架构设计
```
gensim/
├── parsing/      # 文本解析
├── corpora/      # 语料库处理
├── matutils/     # 数学工具
├── interfaces/   # 接口定义
├── models/       # 核心模型（LDA, word2vec, Doc2Vec 等）
├── similarities/ # 相似度计算
└── utils/        # 工具函数
```

### 与涌现实验的潜在结合
1. **主题演化分析** - 用 LDA 分析实验日志的主题变化
2. **相似实验检索** - 用 Doc2Vec 找到相似的实验配置
3. **参数空间探索** - 用嵌入向量分析参数空间的语义结构
