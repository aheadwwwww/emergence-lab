## 21:59 探索：mewo2/terrain — 程序化地形生成
- 克隆并阅读了 terrain.js
- 核心流水线：Voronoi网格 → Gaussian山脉 → 松弛 → 尖峰归一化 → 水力侵蚀 → 海平面 → 汇填充 → 海岸清理
- 河流：由 downhill() + getFlux()（按高度排序的积水优先队列）生成
- 领土分割：基于优先队列的加权 Dijkstra，从城市向外扩展
- 城市放置：河流通量 √ + 边界惩罚 + 现有城市距离惩罚的评分函数
- 所有操作在 Voronoi 网格（非网格）上进行，提供自适应分辨率
- 输出：海岸线等高线、河流、边界、坡度笔触、城市标签、区域标签
- 标签布局：基于贪心惩罚的候选位置评分
