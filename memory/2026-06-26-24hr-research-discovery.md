# 24小时深度研究Agent 学习笔记

> 发现时间：2026-06-26
> 来源：GitHub搜索 autonomous agent experimental

---

## 一、项目概述

**24 Hour Deep Research Agent** 是一个实验性自主研究系统：
- 进行多小时研究会议
- 生成书长度报告（数万字）
- 完整引用管理
- 支持任何主题

**核心价值**：
- 自动分解复杂主题为10-200+研究任务
- 多小时持久化状态
- 多源搜索集成
- 自动引用追踪

---

## 二、核心特性

### 2.1 架构设计

```
Planner Agent (创建计划)
    ↓
Researcher Agent (深度研究)
    ↓
Editor Agent (编译报告)
```

**状态数据库**（SQLite）：
- Tasks（任务）
- Sources（来源）
- Glossary（术语表）
- Sessions（会话）

### 2.2 功能特性

**全面研究**：
- 自动分解复杂主题为10-200+研究任务
- 智能任务优先级（先引言，再正文，最后结论）

**长时间运行**：
- 设计用于多小时研究
- 完整状态持久化
- 可恢复中断的会话

**多源搜索**：
- Tavily（带完整内容提取）
- Serper
- Brave Search API

**引用管理**：
- 自动追踪所有来源
- 内联引用
- 源验证和质量评分

**多格式导出**：
- Markdown
- HTML（专业白色/黑色主题）
- PDF

## 三、示例输出

## 五、关键洞察

**核心价值**：
- 自动化"文献综述"这个耗时过程
- 从主题到报告，无需人类干预
- 可恢复、可追溯、可验证

**我的启发**：
- 三Agent架构比单Agent更高效
- 状态持久化是长时间任务的关键
- 智能任务优先级提高输出质量

**与树林语料的关联**：
- 树林说"一件事一件事做" → 24hr-agent的串行任务执行
- 树林说"从0到1最难，之后就是低成本重复" → Planner创建模板，Researcher重复填充
- 树林说"外显化" → HTML/Markdown/PDF多格式输出
