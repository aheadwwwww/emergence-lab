选择一个模型开始学习

查理芒格说：手里只有锤子的人，看什么都像钉子。
掌握多元思维模型，用不同的透镜看世界。

决策

第一性原理

拆解到最基本事实，从零重建

描述
把问题拆到不能再拆的基本元素，抛弃所有假设和惯例，从最底层重新构建解决方案。马斯克用这个方法把火箭成本降到1/10。
何时使用
当被告知不可能时；当行业惯例成为枷锁时；当需要突破性创新时
思考步骤
  1. 1. 明确你要解决的问题是什么
  2. 2. 拆解：这个问题由哪些基本元素构成？
  3. 3. 质疑：哪些是事实，哪些只是惯例？
  4. 4. 重建：从基本事实出发，能有什么新方案？
决策

二阶思维

思考后果的后果

描述
不只考虑直接结果，还要想'然后呢？'。第一阶：降价导致销量上升。第二阶：竞品跟进导致利润都下降导致行业洗牌。
何时使用
做战略决策时；评估政策影响时；投资判断时
思考步骤
  1. 1. 列出决策的直接后果
  2. 2. 对每个后果问会发生什么
  3. 3. 重复至少两次
  4. 4. 比较一阶和二阶的差异
决策

逆向思维

从失败倒推

描述
不问怎么成功，问怎么失败然后避免。芒格说：告诉我我会死在哪，我永远不去那里。
何时使用
风险评估；项目启动前；重大决策前
思考步骤
  1. 1. 定义你想要的结果
  2. 2. 反过来问：怎样确保达不到？
  3. 3. 列出所有失败路径
  4. 4. 逐一设计防范措施
决策

思想实验

在脑子里做实验

描述
在大脑中模拟场景，探索假设和可能性。爱因斯坦用思想实验发现了相对论。
何时使用
无法做真实实验时；探索如果会怎样；测试假设
思考步骤
  1. 1. 设定场景和假设条件
  2. 2. 在脑中推演事件发展
  3. 3. 观察矛盾点和意外发现
  4. 4. 提炼可验证的结论
决策

概率思维

用概率而非确定论看世界

描述
未来不是确定的，是一组概率分布。好的决策是选择胜率更高的选项，而非保证赢的选项。
何时使用
面对不确定性；评估风险；做预测
思考步骤
  1. 1. 列出所有可能的结果
  2. 2. 估算每种结果的概率
  3. 3. 计算期望值
  4. 4. 选择期望值最高的方案
决策

能力圈

知道边界在哪比圈子多大更重要

描述
巴菲特和芒格的核心原则。不在能力圈外做决策。圈的大小不重要，知道边界在哪才重要。
何时使用
做投资决策；接受新任务；给别人建议
思考步骤
  1. 1. 问自己：这件事我真的懂吗？
  2. 2. 区分知道和以为自己知道
  3. 3. 在圈外时承认不懂找懂的人
  4. 4. 持续扩大能力圈
决策

地图非领土

模型不等于现实

描述
任何理论、模型、计划都只是现实的简化表示。地图再精确也不是领土本身。不要爱上自己的模型。
何时使用
制定计划时；使用理论框架时；分析数据时
思考步骤
  1. 1. 确认你用的是哪个地图
  2. 2. 问：这个地图忽略了什么？
  3. 3. 去领土上实地验证
  4. 4. 根据现实反馈更新地图
决策

奥卡姆剃刀

最简单的解释往往是对的

描述
在多个解释中，选择假设最少的那个。不是简单的一定对，而是不必要的复杂通常是错的。
何时使用
面对多个解释时；设计方案时；调试问题时
思考步骤
  1. 1. 列出所有可能的解释
  2. 2. 数每个解释需要多少假设
  3. 3. 优先验证假设最少的
  4. 4. 如果不够再逐步增加复杂度
决策

汉隆剃刀

能用愚蠢解释的，不要归咎于恶意

描述
别人做了让你不爽的事，先假设他是疏忽或无知，而不是故意害你。大多数恶意其实是笨拙。
何时使用
被人冒犯时；解读竞争对手行为；处理冲突
思考步骤
  1. 1. 暂停：感受到被冒犯时先停下来
  2. 2. 列出其他可能的解释
  3. 3. 问：有证据证明是恶意吗？
  4. 4. 没有证据就选最善意的解释
经济学

复利

利滚利，时间是最好的杠杆

描述
收益产生收益，形成指数增长。不只是钱，知识、技能、人际关系都遵循复利。关键是持续投入和时间。
何时使用
投资理财；学习规划；职业发展
思考步骤
  1. 1. 找到能产生复利的事
  2. 2. 尽早开始让时间发挥作用
  3. 3. 保持持续投入不要中断
  4. 4. 耐心等待指数拐点
经济学

机会成本

选择的真正代价是你放弃的东西

描述
做一件事的成本不是花的时间和钱，而是用同样的资源能做的最好的另一件事。
何时使用
做选择时；分配资源时；评估时间投入
思考步骤
  1. 1. 列出你正在放弃的选项
  2. 2. 找出其中价值最高的
  3. 3. 问：选A放弃B值得吗？
  4. 4. 用放弃的最好选项来衡量成本
经济学

安全边际

留足缓冲应对意外

描述
工程师造桥不造刚好承重的桥，而是留几倍余量。投资、计划、承诺都应该留安全边际。
何时使用
做预算；承诺截止日期；投资决策
思考步骤
  1. 1. 估算最坏情况需要什么
  2. 2. 在估算基础上加缓冲30%到100%
  3. 3. 按加了缓冲的标准执行
  4. 4. 永远不要刚好卡在极限上
经济学

供需关系

价格由供需决定不是由成本决定

描述
东西值多少钱不取决于你花了多少成本，取决于有多少人想要、有多少人提供。
何时使用
定价；分析市场；理解价格波动
思考步骤
  1. 1. 识别谁是供给方谁是需求方
  2. 2. 分析供给和需求各自的弹性
  3. 3. 预测什么因素会改变供给或需求
  4. 4. 判断价格会往哪个方向走
经济学

稀缺性

有限的资源无限的需求

描述
经济学的基本前提。因为稀缺所以需要选择，因为选择所以有机会成本。稀缺创造价值。
何时使用
资源分配；优先级排序；理解价值
思考步骤
  1. 1. 识别什么是真正稀缺的
  2. 2. 区分稀缺和短暂短缺
  3. 3. 问：谁控制了稀缺资源？
  4. 4. 思考如何创造或获取稀缺资源
经济学

激励

人会对激励做出反应

描述
芒格说：告诉我激励是什么，我告诉你结果是什么。设计错误的激励会产生错误的行为。永远先看激励结构。
何时使用
设计制度；管理团队；理解行为
思考步骤
  1. 1. 找出系统中所有的激励
  2. 2. 问这些激励会导致什么行为
  3. 3. 检查有没有激励在鼓励坏行为
  4. 4. 调整让激励和想要的结果对齐
心理学

确认偏误

人只看到自己想看到的

描述
我们倾向于寻找、注意、记住支持自己观点的证据，忽略反对的证据。这是最普遍也最危险的认知偏差。
何时使用
做判断时；收集信息时；评估证据时
思考步骤
  1. 1. 明确自己当前的信念是什么
  2. 2. 主动寻找反对的证据
  3. 3. 问如果我是错的会是什么原因
  4. 4. 让持不同观点的人来挑战你
心理学

社会认同

别人怎么做我就怎么做

描述
在不确定时人会参考他人的行为。这是为什么排队、从众、潮流会形成。也是为什么泡沫会膨胀。
何时使用
看到大家都在做某事时；做独立判断时；分析趋势
思考步骤
  1. 1. 意识到我是不是在跟风
  2. 2. 问如果没人这么做我还会做吗
  3. 3. 找反例有没有人不跟风也成功了
  4. 4. 独立评估这件事本身的价值
心理学

沉没成本

已经花掉的不应该影响未来决策

描述
钱花出去了时间投入了，但这些不应该影响要不要继续。决策只看未来不看过去。
何时使用
考虑是否放弃时；评估项目时；投资决策
思考步骤
  1. 1. 区分哪些是已经花掉的沉没成本
  2. 2. 问如果今天从零开始我会做这个选择吗
  3. 3. 只看未来的成本和收益
  4. 4. 如果答案是不果断放弃
心理学

锚定效应

第一个数字决定了你的判断范围

描述
看到第一个价格后后续判断都会围绕它。标价1000打折到500你觉得便宜，但如果一开始标500呢？
何时使用
谈判；定价；做估算
思考步骤
  1. 1. 识别第一个出现的数字是什么
  2. 2. 问如果没有这个锚我会怎么判断
  3. 3. 主动设置自己的锚
  4. 4. 在谈判中先出价
心理学

框架效应

怎么说比说什么更重要

描述
同样的信息换个说法人的反应完全不同。存活率90%和死亡率10%是一回事但感受截然不同。
何时使用
沟通；说服；做选择时
思考步骤
  1. 1. 识别当前的表述框架
  2. 2. 换个框架重新描述同一件事
  3. 3. 问不同框架下我的判断会变吗
  4. 4. 选择最能帮助理性决策的框架
心理学

损失厌恶

失去的痛苦大于得到的快乐

描述
失去100块的痛苦大约是得到100块快乐的两倍。这导致人过度保守不愿放弃不愿冒险。
何时使用
做风险决策；考虑改变；评估投资
思考步骤
  1. 1. 识别我是不是因为怕失去而在回避
  2. 2. 量化失去和得到的实际价值
  3. 3. 问如果这已经是失去的状态我会花多少挽回
  4. 4. 用理性计算代替情绪反应
系统思维

反馈循环

输出反过来影响输入

描述
正反馈：A增加导致B增加导致A更增加。负反馈：A增加导致B减少导致A被抑制。系统行为由反馈结构决定。
何时使用
分析系统行为；设计流程；理解增长或衰退
思考步骤
  1. 1. 画出系统的因果链条
  2. 2. 找出所有反馈回路
  3. 3. 判断每条回路是正反馈还是负反馈
  4. 4. 问主导回路是什么它会把系统带向哪里
系统思维

涌现

整体大于部分之和

描述
简单规则在大量交互后产生复杂行为。蚁群没有CEO但能建复杂的巢。意识从神经元中涌现。
何时使用
理解复杂系统；设计规则；分析群体行为
思考步骤
  1. 1. 识别系统的微观规则是什么
  2. 2. 观察宏观层面出现了什么模式
  3. 3. 问这个模式能从微观规则推导出来吗
  4. 4. 如果不能这就是涌现
系统思维

瓶颈

系统最慢的环节决定整体速度

描述
一条流水线的速度由最慢的工位决定。优化非瓶颈环节是浪费。找到瓶颈解决它然后找下一个。
何时使用
优化流程；提高效率；诊断问题
思考步骤
  1. 1. 画出整个流程
  2. 2. 找出最慢最堵的环节
  3. 3. 集中资源解决这个环节
  4. 4. 解决后新的瓶颈会出现重复此过程
系统思维

杠杆点

小改变大影响

描述
系统中有些点稍微用力就能产生巨大变化。找到这些点比盲目努力重要100倍。
何时使用
推动变革；解决问题；提高效率
思考步骤
  1. 1. 理解系统的结构
  2. 2. 列出所有可以干预的点
  3. 3. 评估每个点的杠杆效应
  4. 4. 优先干预杠杆最大的点
系统思维

路径依赖

历史决定未来

描述
一旦走上某条路就很难换。QWERTY键盘不是最优布局但所有人都习惯了。选择初始路径时要格外谨慎。
何时使用
做初始选择；设计标准；理解行业格局
思考步骤
  1. 1. 识别当前状态是哪些历史选择造成的
  2. 2. 问如果重来我会选这条路吗
  3. 3. 评估转换成本有多高
  4. 4. 在起点时多花时间选对路
系统思维

熵增

封闭系统必然走向无序

描述
热力学第二定律。不持续输入能量或信息任何系统都会退化。这就是为什么维护比建设更难。
何时使用
理解衰退；维护系统；组织管理
思考步骤
  1. 1. 识别系统是封闭的还是开放的
  2. 2. 测量无序度在增加吗
  3. 3. 找到负熵源什么在对抗熵增
  4. 4. 持续注入能量信息和秩序
生物学

演化

变异加选择加时间等于适应

描述
不需要设计师不需要目标。随机变异环境选择适者生存。这是地球上最强大的优化算法。
何时使用
理解变化；设计迭代流程；接受不确定性
思考步骤
  1. 1. 创造变异尝试不同的方案
  2. 2. 设置选择标准什么算好
  3. 3. 保留好的淘汰差的
  4. 4. 重复很多次
生物学

生态位

找到你独有的位置

描述
每个物种都有自己的生态位。商业也一样找到别人做不了或不愿做的位置。
何时使用
职业规划；商业定位；竞争策略
思考步骤
  1. 1. 分析环境有什么未被满足的需求
  2. 2. 评估自己有什么独特优势
  3. 3. 匹配优势和需求的交集就是你的生态位
  4. 4. 深耕在生态位里做到最好
生物学

合作

一起做比单干强

描述
演化不只是竞争合作同样重要。共生、互惠、协作自然界充满了合作的例子。
何时使用
团队协作；商业合作；生态设计
思考步骤
  1. 1. 识别谁和你有互补的能力
  2. 2. 设计如何让合作双方都受益
  3. 3. 建立信任从小合作开始
  4. 4. 扩展成功的合作模式可以复制
生物学

自催化

产物加速自己的生产

描述
化学反应中产物本身是催化剂。商业中用户越多产品越好导致产品越好用户越多。这是指数增长的底层机制。
何时使用
理解增长；设计产品；分析网络效应
思考步骤
  1. 1. 识别什么在加速自己的增长
  2. 2. 问增长是自催化的还是外部驱动的
  3. 3. 如果是自催化找到加速循环的关键环节
  4. 4. 小心自催化也会加速崩溃
其他

临界质量

过了某个点变化不可逆

描述
核反应需要足够的铀堆在一起才能链式反应。社会运动、技术采用、市场引爆都有临界质量。
何时使用
推动变革；产品发布；理解趋势
思考步骤
  1. 1. 定义什么状态算引爆
  2. 2. 估算需要多少积累才能到临界点
  3. 3. 集中资源推过临界点
  4. 4. 过了临界点后系统会自我维持
其他

林迪效应

存在越久的东西未来存在越久

描述
一本书卖了100年大概率还会再卖100年。时间是质量的过滤器。
何时使用
选择学习内容；判断趋势；做长期决策
思考步骤
  1. 1. 问这个东西存在多久了
  2. 2. 存在越久未来预期寿命越长
  3. 3. 优先选择经得起时间考验的东西
  4. 4. 对新事物保持谨慎
其他

帕累托法则

80%的结果来自20%的原因

描述
80%的销售额来自20%的客户。80%的成果来自20%的努力。找到那20%把精力集中在那里。
何时使用
优先级排序；资源分配；提高效率
思考步骤
  1. 1. 列出所有投入和产出
  2. 2. 找出贡献最大的20%
  3. 3. 把80%精力放在这20%上
  4. 4. 对剩下的80%减少投入或放弃
