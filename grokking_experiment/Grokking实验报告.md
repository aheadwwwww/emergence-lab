# Grokking实验：观察神经网络的"顿悟"现象

## 什么是Grokking？

Grokking是深度学习中一个令人惊叹的现象：神经网络在长时间过拟合训练集后，突然实现了完美的泛化——就像"顿悟"一样！

这个现象由OpenAI团队在2022年的论文《Grokking: Generalization Beyond Ovll Algorithmic Datasets》中首次系统研究。

## 实验设计

### 任务选择：模运算

我们选择了经典的模运算任务来观察Grokking：
- **模数**: P = 97
- **训练集比例**: 30%（只有2822个样本，远少于全部9409个样本）
- **任务类型**: 加法运算和乘法运算

这种设计让模型必须真正理解"模运算"的规则才能泛化，而不是简单记忆。

### 网络架构

```
Embedding(97 → 128)  # 将数字嵌入到高维空间
→ MLP(128 → 128, 2层)  # 处理特征
→ Linear(128 → 97)  # 输出预测结果
```

关键设计：使用两个操作数的嵌入向量的**元素乘积**作为输入，这有助于学习乘法类运算。

### 训练参数

- 学习率: 1e-3
- Weight Decay: 1.0（强正则化是观察到Grokking的关键！）
- 训练轮数: 10,000 epochs
- Batch Size: 512

## 实验结果

### 加法运算 (a + b mod 97)

| 阶段 | Epoch | 训练准确率 | 测试准确率 |
|------|-------|-----------|-----------|
| 初始 | 0 | 1.20% | 0.82% |
| 过拟合 | 1000 | **100%** | 31.47% |
| 过拟合 | 2000 | **100%** | 35.11% |
| **顿悟！** | 3000 | **100%** | **92.58%** |
| 完美泛化 | 4000+ | **100%** | **100%** |

Grokking发生在epoch 3000！模型从测试准确率35%突然跃升至92%+。

### 乘法运算 (a × b mod 97)

| 阶段 | Epoch | 训练准确率 | 测试准确率 |
|------|-------|-----------|-----------|
| 初始 | 0 | 0.92% | 1.03% |
| 过拟合 | 1000 | **100%** | 31.29% |
| 过拟合 | 2000 | **100%** | 48.00% |
| **顿悟！** | 2600 | **100%** | **90.82%** |
| 完美泛化 | 3000+ | **100%** | **99.98%** |

乘法Grokking更早发生（epoch 2600），这可能因为乘法运算有更丰富的结构信息。

## 关键发现

### 1. Grokking的三阶段

```
阶段1: 记忆期（0-1000 epochs）
   - 模型快速记忆训练数据
   - 训练准确率 → 100%
   - 测试准确率 ≈ 30%（仅靠统计规律猜测）

阶段2: 过拟合期（1000-2500 epochs）
   - 继续过拟合训练集
   - 测试准确率缓慢上升
   - 模型内部正"真正"的算法

阶段3: 顿悟期（2500-3000 epochs）
   - 突然实现泛化！
   - 测试准确率 → 90%+
   - Weight Decay促使模型找到简洁的算法解
```

### 2. Weight Decay是关键

没有强正则化，模型会停留在记忆阶段。Weight Decay迫使模型寻找更简洁的解——即真正的模运算算法。

### 3. 模型真的学会了算法

最终测试结果显示：
- **完整测试集准确率: 100%**
- 模型能正确预测所有未见过的组合
- 例如：75 × 39 mod 97 = 15 ✓

## 可视化

训练过程可视化显示了典型的Grokking曲线：

**Loss曲线特点**：
1. 训练loss快速下降至接近0
2. 测试loss先上升（过拟合），然后在顿悟点急剧下降
3. 最终训练和测试loss都收敛到极低值

**Accuracy曲线特点**：
1. 训练准确率快速达到100%
2. 测试准确率长时间停留在30-50%
3. 顿悟点：测试准确率突然跃升至90%+
4. 最终两者都达到100%

## 启示

1. **不要过早停止训练**：即使训练集已经100%准确，继续训练可能带来泛化能力
2. **正则化很重要**：Weight Decay帮助模型找到简洁的算法解，而不是记忆
3. **小数据集也能学算法**：只需要30%的数据，模型就能学会完整的模运算规则
4. **神经网络的"理解"**：Grokking表明神经网络确实能学习真正的算法，而不只是统计模式

## 总结

Grokking揭示了深度学习的深层机制：神经网络在适当条件下会从"死记硬背"进化为"真正理解"。这为理解神经网络的学习过程提供了重要线索，也为训练策略优化提供了新思路。
