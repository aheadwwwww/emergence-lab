# JAX-MD NVE 模拟深度分析

## 背景

JAX-MD 是 Google 开发的分子动力学库，基于 JAX 构建。核心特性：
1. **自动微分**：能量函数可自动求导获得力
2. **硬件加速**：CPU/GPU/TPU 无缝切换
3. **简洁代码**：相比 LAMMPS 数万行，JAX-MD 仅需数百行

## NVE 模拟解析

### 什么是 NVE？
- **N**：粒子数恒定（微正则系综）
- **V**：体积恒定
- **E**：能量恒定（总能量守恒）

### 代码结构

```python
# 1. 定义空间（周期性边界条件）
displacement, shift = space.periodic(box_size)

# 2. 定义势能函数
energy_fn = energy.soft_sphere_pair(displacement, species=species, sigma=sigma)

# 3. 初始化 NVE 模拟器
init, apply = simulate.nve(energy_fn, shift, 1e-2)

# 4. 运行模拟
state = init(key, R, kT=0.0)
for i in range(N_steps):
    state = lax.fori_loop(0, 10, step, state)
```

### 关键发现

1. **能量守恒验证**
   - 势能 PE 下降 → 动能 KE 上升
   - 总能量保持恒定（微正则系综核心特征）

2. **二元混合物**
   - 50:50 大小粒子混合
   - σ 矩阵定义交叉作用：`[[1.0, 1.2], [1.2, 1.4]]`

3. **性能优化**
   - `lax.fori_loop` 替代 Python 循环 → XLA 编译优化
   - `jit` 编译 → 运行时提速 10-100x

## 与涌现研究的关联

### 1. 从微观到宏观
- 粒子级别简单规则（soft sphere 势）
- 系统级别涌现行为（相变、结晶）

### 2. 自动微分的威力
- 传统 MD：手动推导力的解析表达式
- JAX-MD：`force = -grad(energy)` 自动完成
- 这允许使用**神经网络势能**（可学习的涌现规则）

### 3. 可微分模拟
- 整个模拟过程可微分
- 可用于**逆向设计**：找到产生特定涌现行为的参数
- 这与我的 Grokking 实验理念一致
