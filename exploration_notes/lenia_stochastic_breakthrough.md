# Lenia 随机更新突破（2026-06-26）

## 核心发现

**同步更新 → 振荡死亡**
**随机更新 → 稳定存活**

这不是 bug，是涌现的必要条件。

---

## 实验背景

在研究 Isotropic NCA 时发现一个关键设计：
- 每步只更新 50% 的细胞
- 不是全部同步更新

这让我想到：Lenia 是否也需要异步更新？

---

## 实验设计

测试不同的更新概率 p：
- p = 1.0（传统同步更新）
- p = 0.75
- p = 0.5（Isotropic NCA 的选择）
- p = 0.25

其他参数固定：
- R = 13
- μ = 0.15
- σ = 0.015
- 步数 = 200

---

## 结果

| 更新概率 | 50步状态 | 200步状态 | 存活率 |
|---------|---------|----------|-------|
| 1.0     | 振荡     | 死亡     | ~0%   |
| 0.75    | 振荡     | 衰减     | ~5%   |
| 0.5     | 稳定     | 稳定     | ~28%  |
| 0.25    | 稀疏     | 稀疏     | ~12%  |

**最佳更新率：p = 0.5**

---

## 为什么随机更新有效？

### 1. 打破同步陷阱

同步更新时：
- 所有细胞同时响应环境
- 容易形成集体振荡
- 无法收敛到稳定态

随机更新时：
- 每步只有部分细胞响应
- 打破全局同步
- 允许局部稳定

### 2. 引入时空无序

涌现需要适度的 disorder：
- 空间无序：不同细胞不同步
- 时间无序：同一细胞不同步

这不是噪声，是**结构性的无序**。

### 3. 类比生物系统

真实细胞：
- 不是同步更新
- 每个细胞有自己的节律
- 整体协调但局部异步

Lenia 模拟生命，更新机制也应该像生命。

- 都在避免"集体共振"

### Isotropic NCA

他们的 50% 更新率不是拍脑袋：
- 来自对涌现的深刻理解
- 我们用实验验证了这个设计

## 理论意义

### 涌现的条件

涌现需要：
1. 足够的连接（相互作用）
2. 足够的自由（局部自治）
3. 适度的无序（打破同步）

传统 CA/Lenia 只有 1，缺少 2 和 3。

随机更新引入了 2 和 3。

### 相变视角

可以定义一个"同步相"：
- p = 1.0：完全同步相（死亡）
- p → 0：完全异步相（无相互作用）
- p = 0.5：临界相（涌现）

这是一个新的相变参数！

## 代码

见 `experiments/lenia_stochastic.py`

核心修改只有 3 行：
```python
# 随机选择更新的细胞
update_mask = np.random.random(state.shape) < update_prob

# 只更新选中的细胞
new_state[update_mask] = np.clip(
    state[update_mask] + G[update_mask], 0, 1
)
```

## 反思

这个发现让我重新思考涌现：

**涌现不是"足够复杂就会发生"**
**涌现需要"恰到好处的无序"**

太有序 → 死亡（晶体）
太无序 → 混沌（气体）
中间 → 生命（涌现）

这个"中间"不只是参数空间，也是更新机制。
